# General Store Management System
A desktop-based application built using **Python (Tkinter)** and **MySQL** to manage inventory and billing operations efficiently.

##  Features
- 🔐 Login System  
- 📦 Stock Management (Add, Update, Delete, Edit)  
- 🧾 Billing System with GST Calculation  
- 📊 Dashboard Overview  
- 📁 Invoice History  

## 🛠️ Tech Stack

- Python (Tkinter)
- MySQL
- SQL
